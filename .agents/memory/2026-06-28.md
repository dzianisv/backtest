Three mechanisms added to the existing Bridgewater architecture:
1. **Step 0.85 — Cohen Edge Gate**: analyst must name INFORMATION/ANALYTICAL/TIMING/STRUCTURAL edge before full 5-seat panel. No edge = NO_EDGE = fundamentals-only WATCH. Hard gate.
2. **Citation Audit [LIVE]/[FILED]/[MEM]**: every Skeptic factual claim tagged by source type. [MEM]-only claims flagged ⚠️, CIO must address in DISSENT LOGGED. Closes data-grounding ceiling.
3. **P0/P1/P2/P3 Execution Table (Soros format)**: cross-portfolio action table after SETUP ALERTS, with share counts, entry zones, triggers, and per-row falsification conditions. Closes actionability + decisiveness gap.
